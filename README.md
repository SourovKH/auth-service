1. bcryptjs
2. cors
3. dotenv
4. express
5. jwt
6. mongoose
7. morgan
8. nodemailer
9. otp-generator

login process work is also in progress. Yet it's not fully completed few developments are ongoing but after completing this project, can be a well mannered and scalable authenticator system.
